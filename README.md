# OpEnUV – Open Source Extreme Ultraviolet Lithography Simulator

**Open Source. GPU-native. Modular. EUV-Lithographie für alle.**

OpEnUV ist ein Open-Source-Simulator (Apache-2.0) für extreme ultraviolet (EUV) Lithographie.
Entwickelt für Bildung, Forschung und Chip-Design – von der Plasmaquelle bis zur CD-Metrologie.
